# Пару Пар

Каркас сайта банного фестиваля на Vite.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Где менять контент

- Основные тексты, даты, города, мастера, ведущие и галерея: `src/content.js`
- Изображения и логотип: `public/assets`
- Визуальная стилистика: `src/styles.css`

## Примечание

Сайт сделан как самостоятельный фестивальный лендинг с похожей логикой больших ивент-сайтов: крупный герой, карточки городов, программа, участники, медиа и билеты. Он не копирует исходники, верстку или бренд VK Fest.
