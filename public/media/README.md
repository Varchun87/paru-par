# Media placeholders

Загрузите сюда финальные файлы и оставьте такие имена или поменяйте пути в `src/content.ts`.

- `logo-red.png` — логотип для шапки.
- `hero-sauna.jpg` — главное фото первого экрана.
- `master-01.jpg`, `host-01.jpg`, `master-02.jpg`, `artist-01.jpg` — участники.
- `gallery-01.jpg` ... `gallery-08.jpg` — фотогалерея.

Для видео можно положить `teaser.mp4` и заменить блок `video-card` в `src/App.tsx` на тег `video`.
